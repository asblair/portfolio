Pin It

A powerful to do app

Mobile App

Run the app from here: Pin It

For more information, check out the project on: GitHub Pin It Project

A simple app with powerful organizational features that can prioritize to dos. Pin to order high priority to dos at the top of your list. Check to order completed items to the bottom of the list. Lists can easily be updated with text input as well as removed with a delete button.
