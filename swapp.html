Swapp

A free market to exchange products and services

Full Stack Project

Check out the deployed app: Swapp

For more information, check out the project on: GitHub Swapp Project

A matching application for users looking to exchange or swap an unused or unwanted item for something else. The idea about this application is based on a free market system where no money is exchanged. However, users can place values on their items or swaps and see available swaps to exchange within a 20% margin of their initial value.

Swapp offers a great UI which makes it easy to navigate. Users need to register and login to use the application and can create a profile. Users are sent an email notification upon successful registration. They can upload, edit and remove their swaps from their catalog. The user needs to upload a swap in order to see availabe swaps where the user can then search and filter available swaps by category, make and recieve swap requests. An email notification is sent to the user when a request is recieved. In the user area of the application, incoming requests can be viewed, here they can decide to accept or decline the request. Upon decline, the item is removed but a repeated request will not be recieved. Upon accept, the swap will move from the requests area to matches. The sender of the the request will be notified by email that their request was accepted. The swaps will be avaible in both user profiles in the matches area. Here, each user can view the others email so they can be in direct contact to make their swap. Finally matches can be removed from their areas. Additionally, users can make dontions through Stripe to support a charity that supports people in need.
