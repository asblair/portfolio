RechargeNow

EV charging points nearby you

Mobile App

Download from the App Store: RechargeNow

For more information, check out the project on: GitHub RechargeNow Project

A useful app for electric car owners looking for a charging point nearby and also a place to have a coffee whilst you wait for your car to be recharged.

This app can display all registered charging points anywhere in UK, Europe and North America within a 500km radius of the user's location. The user can decrease the size of this radius to display charging only within this radius. The user can also search and filter by the power output of the charger. Users can then click on the chrage point icon to display information about it, with the option to save it to favourites, to go back or to get a coffee nearby. The get a coffee nearby then isolates the selected charger point and displays coffee shops within a 100m radius. The use can select the coffee shops to display informations such as full address, if its open or closed and its rating. In favourites, users can see their chosen favourites and can choose list or map view. The user can also remove the favourites from here.
